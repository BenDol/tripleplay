TriplePlay UI

TriplePlay contains a UI framework (called TPUI for short) that provides many standard widgets for use in a PlayN game. This demo shows off some of the widgets and other user interface elements.

It's a little haphazard because it tends to act as a testbed and a demo. If someone wants to create a more organized demo, that would be great!
